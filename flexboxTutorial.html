Flexbox Basics

Flexbox is a CSS display layout method that allows for easy arranging of components in rows or columns. Flexbox is an easy to use, reliable and cross-browser tool for creating a layout for a webpage. It is able to achieve the same functionality as floats and positions while having added features that make a variety of designs easier to implement.
