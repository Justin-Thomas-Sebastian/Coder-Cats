Responsive Images

Responsive images are used to ensure that images are compatible on different devices with differing screen sizes and resolutions. There are many ways to ensure that images in your web page are responsive. A common method is to use HTML tags and attributes that ensure that only the appropriate images are loaded on any given view of the web page. Though JavaScript and CSS can be used to dynamically resize images, this takes extra processing that can make page loading slower.
